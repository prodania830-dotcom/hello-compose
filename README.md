# hello-compose

Навчальний Android-проєкт на Kotlin з використанням Jetpack Compose.

## Що реалізовано

- встановлення Android Studio та SDK
- створення проєкту HelloCompose
- Git репозиторій
- GitFlow структура
- Issue Templates
- Pull Request Template
- CI через GitHub Actions

## Як запустити

1. Відкрити проєкт у Android Studio
2. Дочекатися Gradle Sync
3. Запустити емулятор
4. Натиснути Run
